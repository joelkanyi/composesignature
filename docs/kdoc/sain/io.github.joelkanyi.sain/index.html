enum SignatureAction : Enum<SignatureAction>

SignatureAction is an enum class that represents the actions that can be performed on a signature. CLEAR clears the signature. You can take advantage here and clear the signature from your implementation size. COMPLETE completes the signature. Once the signature is completed, you receive an ImageBitmap of the signature.
