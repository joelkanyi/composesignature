Sain

@Composable
fun Sain(modifier: Modifier = Modifier, state: SignatureState, signatureColor: Color = Color.Black, signatureThickness: Dp = 5.dp, onComplete: (signature: ImageBitmap?) -> Unit, actions: @Composable (action: (SignatureAction) -> Unit) -> Unit)

Sain is a composable that allows the user to draw a signature on the screen. The resulting signature is returned as an ImageBitmap.

Parameters

modifier

The modifier to apply to the signature.

signatureColor

The color of the signature.

signatureThickness

The thickness of the signature. Note that the thickness is multiplied by 3 to make the signature more visible.

onComplete

The callback that is called when the signature is completed. This callback receives a nullable ImageBitmap of the signature and can be utilized to check if the signature is empty.

actions

The composable that provides the actions that can be performed on the signature. We only have two actions: SignatureAction.CLEAR and SignatureAction.COMPLETE.
